# conversational-analytics-at-home

A small demo project showing how to creaate a Conversational Analitics Tool using [Streamlit](https://streamlit.io/) and [LangChain](https://www.langchain.com/)

Before running this project, download the [NBA Dataset](https://medium.com/r/?url=https%3A%2F%2Fwww.kaggle.com%2Fdatasets%2Fwyattowalsh%2Fbasketball) to the *data* folder.

To start the app run this:
```
streamlit run src/app.py
```
